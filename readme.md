# mojidet

!!! Before use
1. install nodejs 
  <a>https://nodejs.org/dist/v11.0.0/node-v11.0.0-x64.msi</a>
2. install gulp globally
  <b>npm install gulp -g</b>

#1 Install node dependencies <br>
  <b>npm i</b>

#2 Start development test <br>
  <b>gulp watch</b>
  
#3 Package for release <br>
  <b>gulp build</b>
